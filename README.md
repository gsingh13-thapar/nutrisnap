# NutriSnap - Smart Food Diary & Macro Tracker

NutriSnap is a comprehensive, AI-powered food tracking application designed to help fitness enthusiasts and health-conscious individuals log their meals effortlessly.

## Features

- **Smart Analysis:** Intelligent image parsing to automatically detect meals and generate highly accurate macronutrient profiles (Calories, Protein, Carbs, Fat, Fiber).
- **Gym Recipes:** Add, discover, and save gym-focused recipes featuring embedded YouTube tutorials and prominent macro displays.
- **Interactive Dashboard:** Visualize your daily intake vs. fitness goals with beautiful, real-time charts and metrics.
- **Food Diary:** Keep a detailed, time-stamped log of everything you consume categorized by meal type.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, shadcn/ui, Vite
- **Backend:** Firebase (Auth, Firestore)
- **Icons & UI:** Lucide React, Framer Motion (via shadcn)

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Configure your Firebase credentials in `.env`
4. Run `npm run dev` to start the development server
